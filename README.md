# Second Order - News Generator

This project automates the creation of faceless geopolitical analysis videos.

## Workflow
1. Input topic
2. Generate script using AI
3. Convert to video using VideoGen
4. Export and upload

## Prompt Template

Create a 3-minute geopolitical analysis video script.

Structure:
- Hook
- What happened
- What it means
- What happens next

Tone:
- Sharp
- Analytical
- No fluff
